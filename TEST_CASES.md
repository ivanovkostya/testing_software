## ТК-03: Ввод номера карты в поле ввода на форме перевода

| Элемент | Описание |
|---------|----------|
| **Приоритет** | Medium |
| **Тип** | Позитивный |
| **Автор** | Тестировщик |

### Предусловия
1. Приложение запущено по адресу `http://localhost:8000`
2. Выбрана любая валюта (например, «Рубли»)

### Шаги
1. Кликнуть в поле «Номер карты»
2. Ввести 16 цифр подряд (например, `1234567890123456`)
3. Продолжить ввод после 16 цифр (например, ввести ещё 4 цифры)

### Ожидаемый результат
1. Введённые цифры автоматически группируются по 4 цифры: `1234 5678 9012 3456`
2. В поле содержится не более 16 цифр (без учёта пробелов)
3. Дополнительные символы после достижения 16 цифр не добавляются

### Постусловия
Очистить поле ввода

### Связанный автотест
`test_03_card_input`
